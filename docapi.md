## 🏷️ Category Endpoints

### 1. Get All Categories
**GET** `/api/category`

Mendapatkan daftar semua kategori.

#### Response Success (200)
```json
{
  "success": true,
  "data": [
    {
      "id": "number",
      "category": "string",
      "created_at": "datetime"
    }
  ],
  "total": "number"
}
#### Response Error
- **500 Internal Server Error**: Gagal mengambil data kategori
